# Photon - Image Viewer (V1.0)

A image viewer built with C and SDL2.


## Quick Start

### Install Dependencies

**Windows (MSYS2):**
```bash
pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-SDL2 mingw-w64-x86_64-SDL2_image
```

### Build
```bash
git clone https://github.com/yourusername/photon.git
cd photon
make
```

### Run
```bash
./photon image.jpg
```

## Controls

| Key | Function |
|-----|----------|
| ESC | Exit |
| +/- | Zoom |
| F | Fit to window |
| 1 | Actual size |
| I | Toggle info |
| Mouse wheel | Zoom |
